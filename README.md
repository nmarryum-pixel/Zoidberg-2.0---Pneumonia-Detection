# Zoidberg-2.0---Pneumonia-Detection
A deep learning project that detects pneumonia from chest X-rays by training and comparing three CNN models  including a custom CNN, MobileNetV2, and VGG16. Covers the full ML pipeline: EDA, PCA, preprocessing, transfer learning, cross-validation, and threshold tuning. Best model achieves ROC-AUC of 0.958.
